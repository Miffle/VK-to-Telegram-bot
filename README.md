# VK to Telegram bot
## https://t.me/Replacing_vk_messagesbot
## This repository contains the source code of the bot, thanks to which it is possible to receive messages from VK to private Telegram messages.

### List of functions:
1) Receiving messages
2) Reading without the "Read" mark
3) Receiving notifications about a new message
4) Writing and reading messages in one of the 5 best chats
5) Search for a person from the friends list to write a message

Further features in development
